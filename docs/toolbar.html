The toolbar

Go back in the browsing history.

Go forward in the browsing history.

Go to the “Book” branch and expand it. Same as View > Show Book.

Go to the “Characters” branch and expand it. Same as View > Show Characters.

Go to the “Locations” branch and expand it. Same as View > Show Locations.

Go to the “Items” branch and expand it. Same as View > Show Items.

Go to the “Arcs” branch and expand it. Same as View > Show Arcs.

Go to the “Project notes” branch and expand it. Same as View > Show Project notes.

Save the project. Same as File > Save or Ctrl-S.

Toggle the lock status of the project.

Export the manuscript for editing. Same as Export > Manuscript for editing.

Discard the current manuscript, if nes chapters or sections have been inserted. Same as selecting the manuscript under Import.

Add element. Same as Ctrl-N.

Add child element. Same as Ctrl-Alt-N.

Add element on the parent’s level. Same as Ctrl-Alt-Shift-N.

Delete selected elements. Same as Del.

Toggle Text viewer. Same as View > Toggle Text viewer or Ctrl-T.

Toggle Properties. Same as View > Toggle Properties or Ctrl-Alt-T.
