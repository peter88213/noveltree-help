Arc properties

The Arc properties view opens in the right pane when you select an arc in the tree.

Title and description

Title and description are displayed in an editable “index card”.

The title can be completed by pressing the Enter key. Changes to the description are applied when the mouse is clicked anywhere outside the text input field.

Navigation buttons

  • Previous moves the selection to the previous arc in the tree.

  • Next moves the selection to the next arc in the tree.
